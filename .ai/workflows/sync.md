# Workflow: Sync

**Goal**: Ensure the documentation ("The Map") accurately reflects the codebase ("The Territory").

## Protocol Steps

1.  **Index Update**
    * Scan the `src/` directory.
    * Update `docs/protocol/INDEX.md`:
        * Add new files.
        * Remove deleted files.
        * Update descriptions if responsibilities changed.

2.  **Session Summary (Optional)**
    * If finishing a session, you may append a summary to `WORK_LOG.md`.

3.  **Handoff**
    * Wait for user input.
